Listas com HTML5

Listas Ordenadas

Lista onde a ordem dos itens é importante, receitas. passo a passo...

Ex: Tutorial de como executar um arquivo:

  1. Ache o arquivo desejado
  2. Dê dois cliques
  3. Será perguntado: Sim ou Não para executar
  4. Selecione Sim

A tag <ol> possui alguns parâmetros onde podemos configurar o marcador da lista com o parâmetro type:

1 - valor padrão(1,2,3,4,5...)
A - listas com letras maiúsculas(A,B,C,D,E...)
a - listas com letras minúsculas(a,b,c,d,e...)
I - listas com algarismos romanos maiúsculos(I,II,III,IV,V...)
i - lista com algarismos romanos minúsculos(i,ii,iii,iv,v...)

Você também pode indicar de onde a contagem vai começar usando o parâmetro start:
Usando a tag <ol type="I" start = "5"> vai gerar itens como = V, VI, VIII, I,...

Lista Não Ordenada

São listas onde a ordem dos itens não importa, ou seja não tem uma classificação como: primeiro, segundo, terceiro
fazendo com elas se tornem listas aleatorias
Ex: Frutas e Doces:

  • Pudim
  • Banana
  • Bolo
  • Maça

Temos alguns marcadores para a tag <ul>:

disc - padrão, uma bola preta totalmente pintada
circle - uma bola com borda preta e sem preenchimento
square - um pequeno quadrado preto totalmente pintado

Listas e SubListas

Podemos por listas dentro de listas:

  1. Item 1
  2. Item 2
    1. Item 2.1
    2. Item 2.2
    3. Item 2.3
  3. Item 3

Praticando:

  • Nintendinho
    1. Mario Bross
      • Super Mario Bross 3
      • Mario: Lost Levels
    2. Ninja Gaiden
      • Ninja Gaiden III
  • Super Nintendo
    1. Super Mario World
    2. Donkey Kong
      • Donkey Kong Country III
      • Diddy's Kong Quest
  • Playstation 1
    1. Final Fantasy
      • Final Fantasy VII
      • Final Fantasy IX
    2. Castlevania
      • Simphony of the Night
